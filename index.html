What DeFi Assist does

Wallet Tracker Mode

Connect one wallet to a sheet, set update frequency and timezone, and keep portfolio data refreshed automatically.

Automated Portfolio Sync

Collect protocol and token positions, update current value, and maintain a clean portfolio table for analysis.

Key Metrics Engine

Calculate IRR, ROI, P&L, investments, and withdrawals from your transaction history directly inside Sheets.

Cross-chain Transactions

Fetch transaction history by chain, process old and new records, and monitor progress in the sidebar.

Master Dashboard Mode

Aggregate multiple wallet spreadsheets into a single dashboard with imported current value, P&L, and last update.

Privacy-first Approach

No email harvesting. Data is used to run the add-on workflows and sync your DeFi reporting in Google Sheets™.
